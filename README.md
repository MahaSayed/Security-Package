# Security-Package

Computer Security course's project, Implementing Encryption and Decryption Algorithms.

Implemented Algorithms:
------------------------
- AutokeyVigenere
- Ceaser
- Columnar
- HillCipher
- Monoalphabetic
- PlayFair
- RailFence
- RepeatingKeyVigenere
- AES (+ Extended Euclidean)
- DES (+ TripleDES)
- DiffieHellman
- RSA
